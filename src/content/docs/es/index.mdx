# Documentación de Primebrick

Bienvenido a la documentación oficial de **Primebrick**, un framework backoffice de código abierto y opinionado, publicado bajo la **licencia MIT**.

Primebrick ofrece una plataforma completa para construir aplicaciones backoffice con soporte multi-cloud, arquitectura de microservicios y sin vendor lock-in.
