# Documentação do Primebrick

Bem-vindo à documentação oficial do **Primebrick**, um framework backoffice de código aberto e opinativo, distribuído sob a **licença MIT**.

Primebrick oferece uma plataforma completa para construir aplicações backoffice com suporte multi-cloud, arquitetura de microsserviços e sem vendor lock-in.
