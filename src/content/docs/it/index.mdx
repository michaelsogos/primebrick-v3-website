# Documentazione Primebrick

**Primebrick** è un framework backoffice open source opinionato, distribuito sotto licenza **MIT**.
Nato per eliminare il vendor lock-in, Primebrick ti permette di costruire applicazioni backoffice
complete e portarle su qualsiasi cloud — o on-premise — senza riscrivere una riga di codice.

### Riferimento API

La sezione [Riferimento API](/it/api/introduction/) descrive in dettaglio l'API REST di Primebrick,
incluse le convenzioni per autenticazione, autorizzazione, microservizi e gestione degli errori.

- [Introduzione API](/it/api/introduction/) — URL di base e struttura degli endpoint
- [Autenticazione](/it/api/authentication/) — Casdoor IDP, OIDC, JWT e chiavi API
- [RBAC](/it/api/rbac/) — Permessi wildcard e role mappings
- [Standard microservizi](/it/api/microservice-standard/) — Autoregistrazione, heartbeat e proxy
- [Gestione errori](/it/api/error-handling/) — RFC 7807 Problem Details
