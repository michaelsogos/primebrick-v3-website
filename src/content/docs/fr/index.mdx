# Documentation de Primebrick

Bienvenue dans la documentation officielle de **Primebrick**, un framework backoffice open source et opinionated, publié sous la **licence MIT**.

Primebrick offre une plateforme complète pour construire des applications backoffice avec support multi-cloud, architecture de microservices et sans vendor lock-in.
