# Primebrick-Dokumentation

Willkommen in der offiziellen Dokumentation von **Primebrick** – einem opinionated Open-Source-Backoffice-Framework unter der MIT-Lizenz.

Primebrick bietet eine vollständige Plattform für die Erstellung von Backoffice-Anwendungen mit Multi-Cloud-Unterstützung, Microservice-Architektur und ohne Vendor Lock-in.
